# My Game Development Lab 🕹️

Welcome to my collection of interactive games and simulations. This repository showcases my journey through game logic, physics programming, and UI design, primarily using **Scratch** and **Microsoft MakeCode Arcade**.
---

## 🛠️ Concepts & Skills
* **Physics Modeling:** Implementation of $y$-velocity algorithms to simulate realistic free-fall and energy dissipation on bounce.
* **Object-Oriented Logic:** Efficient use of cloning and local/global variables.
* **User Interface:** Designing intuitive controls and interactive sliders.
* **State Management:** Handling "Game Over", "Victory", and "Level Up" states.

## 🎮 How to Run

### For Scratch Projects (`.sb3` files):
1. Download the desired project file.
2. Go to the [Scratch Editor](https://scratch.mit.edu/projects/editor/).
3. Select `File` -> `Load from your computer`.

### For Microsoft Arcade Projects:
1. Open [MakeCode Arcade](https://arcade.makecode.com/).
2. Select `Import` -> `Import File` to load the project.
